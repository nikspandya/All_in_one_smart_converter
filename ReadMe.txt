How to use "All in one file converter"


1. Install python 3.7 or higher and install dependancy from requirements.txt

2. Navigate All_in_one_file_converter folder with smart text editor or IDE 

3. Run app.py from text editir, IDE or form command prompt

4. Open http://127.0.0.1:6008/ from local PC

5. Upload any file with  '.pdf', .jpg', '.png', '.bmp', '.ppm', '.gif', '.tiff' extension

6. Select file type for conversation ('.jpg', '.png', '.bmp', '.ppm', '.gif', '.tiff')

7. Press upload and convert buttom and converted file will be stored in the "/All_in_one_file_converter/static" folder
